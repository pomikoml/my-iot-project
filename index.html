ความหมาย

การที่อุปกรณ์ต่างๆ สิ่งต่างๆ ได้ถูกเชื่อมโยงทุกสิ่งทุกอย่างสู่โลกอินเตอร์เน็ต ทำให้มนุษย์สามารถสั่งการควบคุมการใช้งานอุปกรณ์ต่างๆ ผ่านทางเครือข่ายอินเตอร์เน็ต

ความสำคัญ

ด้านการทำงาน

ช่วยลดในเรื่องของเวลา เช่น การส่งเอกสาร

ด้านความปลอดภัย

ช่วยรักษาความปลอดภัย เช่นกล้องสามารถทำงานได้ตลอดเวลา โดยที่เราไม่ต้องอยู่เฝ้า และมีระบบยืนยันตัวตน เพื่อเข้าใช้งานอีกขั้น

ด้านการแพทย์

การฝังชิฟไว้ในตัวคนไข้ เพื่อที่จะได้รู้ข้อมูลกรุ๊ปเลือด ข้อมูลการเกิดภูมิแพ้ และอื่นๆ

IOT ในด้านการเกษตร อุตสาหกรรม

โรงเรือนปลูกเมล่อน

ปรับลดอุณหภูมิภายในโรงเรือนให้ต่ำลงได้ 4 ถึง 8 °C

พร้อมเชื่อมต่อระบบเซ็นเซอร์วัดผล แบบไร้สาย

สามารถวัดความชื้นในดินและระบบวาล์วน้ำไฟฟ้าแบบไร้สาย

โดยเชื่อมต่ออุปกรณ์ควบคุมดูแล ผ่านแพลตฟอร์มฟาร์มอัจฉริยะ iFram

ทำให้สามารถควบคุมดูแลฟาร์ม ได้ผ่านสมาร์ทโฟน

ได้ผลผลิตเพิ่มเป็น 2 เท่า และมีคุณภาพตามมาตราฐาน

Smart Home

ใช้ Solar cell เมื่อแสงตกกระทบกับสารกึ่งตัวนำจะถ่ายเทพลังงานให้อิเล็กตรอน และโครงสร้างของอะตอมที่สูญเสียอิเล็กตรอน ทำให้เกิดการเคลื่อนไหวซึ่ง อิเล็กตรอนจะเคลื่อนไหวไปที่ Front Electrode และโครงสร้างของอะตอมที่สูญเสียอิเล็กตรอนจะเคลื่อนไปรวมกันที่ Black Electrode เมื่อครบวงจรจะเกิดกระแสไฟฟ้า

Smart Switch รับไฟจากแผง Solar cell เชื่อมเข้ากับสัญญาณ Wi-Fi มี Sensors ตรวจจับ Day Time Night Time เพื่อเปิด-ปิดอัตโนมัติ

Gateway เป็นตัวกลางที่เชื่อมระหว่าง App กับตัวอุปกรณ์ที่ไม่ใช้ Wi-Fi เพราะตัวอุปกรณ์ที่เชื่อมต่อWi-Fi จะคุยกันบนสัญญาณ Internet

Controller เป็นตัวควบคุมในส่วนของ Application สามารถควบคุม Switch ให้เปิด-ปิดได้ตลอดเวลา

Voice Controller สั่งการด้วยเสียงให้ เปิด-ปิด Switch ห้องต่างๆในบ้านได้อย่างง่ายดาย

ใช้งานยังไงให้ปลอดภัย

เชื่อมต่ออินเทอร์เน็ตเมื่อจำเป็น

ทุกครั้งที่มีการเชื่อมต่ออินเทอร์เน็ตต้องมีประโยชน์ และมีความจำเป็นต้องใช้

แยกวงเน็ตเวิร์คออกมาต่างหาก

แยกเป็น Guest Networking เพื่อไม่ให้ผู้ที่ไม่ได้รับอนุญาตเข้าถึงไฟล์หรืออุปกรณ์ได้

ตั้งรหัสผ่านให้ปลอดภัย

ไม่ว่าจะเป็นการตั้งรหัสผ่านที่ยาว และไม่ใช้รหัสผ่านเดียวกันกับเครื่องอื่น สามารถช่วยป้องกัน Hacker ได้

ปิดการใช้ UPnP

เป็นฟีเจอร์ที่ทำให้อุปกรณ์ต่าง ๆ เชื่อมต่อกันได้ง่าย Hacker สามารถใช้ช่องโหว่นี้ในการโจมตี

อัปเดตเฟิร์มแวร์สม่ำเสมอ

เฟิร์มแวร์และแพทช์ล่าสุดเป็นสิ่งที่ควรอัพเดทให้อุปกรณ์อย่างสม่ำเสมอ เพื่ออุดช่องโหว่ก่อนที่ Hacker จะเข้าโจมตี

ระวังการใช้ Cloud Services

การจะใช้ cloud ต้องเชื่อมต่ออินเทอร์เน็ตสู่ภายนอก ทำให้ข้อมูลเกิดการรั่วไหลไปสู่ภายนอกได้

ไม่นำอุปกรณ์ส่วนตัวไปใช้ในที่ทำงาน

หลีกเลี่ยงการนำอุปกรณ์ IoT ส่วนตัวไปใช้ที่ทำงาน เนื่องจากอุปกรณ์เหล่านี้ยังมีประเด็นเรื่องความมั่นคงความปลอดภัยของทุกๆองค์กร

ติดตามตลอดเวลา

ถ้าเจออุปกรณ์ที่ไม่รู้จักให้ทำการ Block ทันที

เทคโนโลยีที่เกี่ยวข้อง

Cloud

มีหน้าที่จัดเก็บข้อมูล หรือนำข้อมูลมานำเสนอในรูปแบบ App server

End nodes

เป็นสิ่งของและอุปกรณ์ตรวจจับ เช่น microcontroller, Single board และ sensors

Connectivity

ทำให้สิ่งของและอุปกรณ์ตรวจจับเชื่อมต่ออินเทอร์เน็ตได้ เพื่อส่งข้อมูลระหว่างอุปกรณ์ไปยัง Cloud

ประโยชน์ของ IOT

ช่วยเพิ่มประสิทธิภาพในการทำงาน

เพิ่มความแม่นยำสามารถเก็บข้อมูล ประมวลผล และแสดงผลได้อย่างรวดเร็ว รองรับข้อมูลได้จำนวนมาก

ไร้ข้อจำกัดด้านเวลาและสถานที่

ไม่มีปัญหาเรื่องเวลาและสถานที่ สามารถทำงานได้ทุกที่ทุกเวลา
